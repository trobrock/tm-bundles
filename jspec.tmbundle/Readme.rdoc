
== JSpec TextMate Bundle

All matcher snippits follow the convention of using the 
first letter of each word, for example:

  should.throw_error : ste
  should.be_an_instance_of : sbaio
  should.match : sm
  ...

== Installation

Copy / paste / execute the following command line to install the bundle

  cd ~/Library/Application\ Support/TextMate/Bundles/ && git clone git://github.com/visionmedia/jspec.tmbundle.git
  
== Notes

* Not all matchers have snippits (yet), feel free to contribute.
